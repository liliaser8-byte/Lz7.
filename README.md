Ця програма призначена для автоматизованої обробки двовимірного масиву (матриці) і візуально демонструє два процеси:
Генерація та обчислення: Програма створює таблицю випадкових чисел, виводить її на екран та математично обчислює суму квадратів усіх елементів цього масиву.
Вибіркове сортування: Програма бере цей масив, виділяє в ньому лише парні стовпці (з індексами 0, 2, 4...) і сортує елементи в них за спаданням (від найбільшого числа зверху до найменшого знизу), залишаючи інші стовпці без змін. Результат виводиться у другу таблицю.
